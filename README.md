## Description

Submit Jokes Microservice - The Jokes are Submit here

# Used technologies
NestJs,
TypeScript,
MongoDB/Mongoose

# Used Tools
NestJs Documentation,
Git,
ChatGPT

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swagger Enabled
http://localhost:<PORT>/api
